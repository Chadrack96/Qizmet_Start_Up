Nos Services

I. Collecte mobile des données avec KoBoToolbox et Kobo Collect

La collecte mobile des données consiste à utiliser des téléphones (principalement
des smartphones ou des tablettes) pour recueillir des données au lieu d’utiliser
des formulaires papier. KoBoToolbox est une plateforme open-source qui permet de créer des questionnaires numériques et de collecter des données via l’application
Android Kobo Collect.

Objectifs :

  • Découvrir l’application KoBoToolbox.
  • Créer un compte KoBoToolbox.
  • Digitaliser des questionnaires.
  • Importer des questionnaires XLSForm.
  • Collecter des données avec l’application KoBo Collect.
  • Gérer les données dans le serveur KoBoToolbox.
  • Visualiser des données en temps réel.
  • Exporter les données et générer des rapports.
  • Prérequis : Connaissances de base en informatique.

II. Analyse statistique avec Excel, Origin, SPSS et R

Ces outils sont utilisés pour analyser des données quantitatives.
Chaque outil a ses avantages et inconvénients, selon vos besoins spécifiques.

Details :

  • Excel : Tableurs pour des analyses simples.
  • Origin : Logiciel de traitement et de visualisation de données scientifiques.
  • SPSS : Logiciel statistique pour l’analyse avancée.
  • R : Langage de programmation statistique open-source.

V. Développement d’applications avec C# et SQL Server

Vous pouvez développer des applications de bureau, des sites web
ou des applications mobiles avec ces technologies.

Avec :

VI. HTML et CSS
